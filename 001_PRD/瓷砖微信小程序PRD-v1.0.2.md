## 1. 产品概述

### 1.1 产品定位

一款面向家装业主、装修工人/包工头、设计师/装修公司三端用户的品牌自营瓷砖微信小程序。以高质量线上展示解决"看不见实物不敢买"的核心痛点，打通从浏览到下单的完整购买链路。

### 1.2 目标用户

| 用户角色 | 特征 | 核心诉求 |
|----------|------|----------|
| 家装业主 | 装修自己房子的普通消费者 | 看清产品细节和铺贴效果，获取导购建议 |
| 装修工人/包工头 | 替业主采购瓷砖 | 快速查找产品，高效下单 |
| 设计师/装修公司 | 为客户选材 | 获取高清素材、CAD文件、独家纹理包和产品规格信息 |

### 1.3 核心价值

**让用户在线上就能看清楚瓷砖，敢下单。**

- 高清产品图 + 纹理细节放大
- 多场景真实装修效果图
- 完整规格参数透明展示
- 真实装修案例参考

## 3. 用户注册与登录

### 3.1 登录流程

微信小程序使用微信官方授权体系，无需用户设置密码。

**首次登录流程：**

```
用户打开小程序
    │
    ▼
检测登录状态
    │
    ├── 已登录 ──────────── 直接进入首页
    │
    └── 未登录
          │
          ▼
    ┌──────────────────┐
    │  品牌展示 + 轮播  │    ← 3 张功能轮播图
    │  "微信授权登录"   │    ← 绿色按钮 #07C160
    └──────┬───────────┘
           │ 用户点击"微信授权登录"
           ▼
    wx.login 获取临时 code
           │
           ▼
    后端用 code 换取 openid / session_key
           │
           ├── 老用户 ────── 生成 token → 进入首页
           │
           └── 新用户
                 │
                 ▼
           ┌──────────────┐
           │  角色选择页    │    ← 3 张大卡片
           └──────┬───────┘
                  │
                  ▼
           ┌──────────────┐
           │  手机号绑定页  │    ← 可跳过
           └──────┬───────┘
                  │
                  ▼
             创建用户 → 进入首页
```

**再次打开（已登录）：**

```
打开小程序 → 静默校验 token → 直接进入首页
```

**页面布局（从上到下）：**

| 区域 | 内容 | 说明 |
|------|------|------|
| 当前绑定 | 已绑定手机号展示 | 显示脱敏手机号 138****6789 + "已验证"徽章 |
| 授权按钮 | "微信手机号一键授权" | 调用微信 getPhoneNumber 组件 |
| 手动修改 | 手机号输入 + 验证码 | 国家码 +86，6 位验证码，发送验证码按钮 |
| 跳过 | "暂不绑定，先逛逛" | 灰色链接，可跳过 |
| 安全提示 | "端到端加密" + "身份已验证" | 底部两张安全卡片，降低用户顾虑 |

**业务规则：**
- 用户选择"暂不绑定"后正常进入首页
- 下单时若未绑定手机号，强制弹出手机号授权
- 手机号通过微信加密数据获取，后端使用 session_key 解密后存储
- 手机号存储需脱敏，前端仅显示 `138****6789` 格式

## 12. 非功能需求

### 12.1 性能

- 图片需压缩优化，首页首屏加载 < 2秒
- 产品图支持 CDN 加速
- 列表页分页加载，每页20条
- 登录鉴权耗时 < 500ms（静默登录）

### 12.2 兼容性

- 支持微信小程序基础库 2.10+
- 适配 iOS 和 Android 两端
- 支持 iPhone SE 至最新型号、主流 Android 机型

### 12.3 安全

- 用户手机号需授权获取，符合微信隐私规范
- 微信 openid 和 session_key 仅在服务端处理，不暴露给前端
- 手机号 AES 加密存储，前端仅展示脱敏格式
- 支付对接微信支付，不自行存储支付敏感信息
- 商家端需账号密码登录，支持 RBAC 权限控制
- 用户注销后个人数据按《个人信息保护法》要求删除

### 12.4 隐私合规

- 用户首次进入时展示隐私协议弹窗，需明确同意后才调用微信授权接口
- 手机号授权独立触发，不与登录授权捆绑
- 小程序设置页提供"隐私设置"入口，用户可查看已授权信息
- 提供账号注销功能（"我的"页面底部），注销后 30 天冷静期，到期后永久删除
